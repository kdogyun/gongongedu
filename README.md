# gongongedu

`setting.ini`을 열어서 아이디와 비밀번호를 자기걸로 바꾸기

```
아이디
비번
timeout_wait (웹페이지 부를때 최대 기다리는 초)
time_sleep (몇초마다 재생이 완료됐는지 확인하는 초)
poll_frequency (웹페이지 로딩이 됐는지 확인하는 주기, 초)
```
